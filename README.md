##MeteoStation

Arduino based weather station. Shows:

- Temperature
- Humidity
- Atmospheric pressure

Hardware:

- DHT AM2302 for temperature and humidity
- BMP180 atmospheric pressure and alternate temperature sensor
- 16x2 LED display
